## What's in it

`epmc` reads Europe PMC through the open EBI REST API. No API key required.
Coverage spans PubMed (MED), PubMed Central (PMC), and preprint servers
(PPR: bioRxiv, medRxiv, ChemRxiv, and others).

### Commands

| command | what it does |
|---|---|
| `epmc search <query>` | Search articles by keyword, author, journal, or any query syntax |
| `epmc article <id>` | Show one article by PMID, PMC ID, or DOI |
| `epmc citations <id>` | Articles that cite a given paper |
| `epmc references <id>` | Articles referenced by a given paper |
| `epmc preprints` | Recent preprints across all servers |

All commands support `-o table|json|jsonl|csv|tsv|url|raw`, `--fields`,
`--template`, `-n/--limit`, `--sort`.

### Sort options for `search`

| flag | order |
|---|---|
| `--sort cited` (default) | most-cited first |
| `--sort date` | newest first |
| `--sort relevance` | API relevance score |

## Install

```bash
go install github.com/tamnd/europepmc-cli/cmd/epmc@v0.1.0
```

Or download a prebuilt binary from the [releases page](https://github.com/tamnd/europepmc-cli/releases/tag/v0.1.0).
